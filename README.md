# NMEA-0183 XDR Sensors on ESP32 m5atomS3-lite

NMEA-0183 XDR Sensors on m5atomS3-lite for Arduino IDE.

The sensors stream NMEA-0183 XDR sentences via USB

## Sensors:

- Temperature
- Humidity
- Air Pressure
- Voltage
- Illumination
